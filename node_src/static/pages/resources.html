All the files above as one zip folder - includes GCODES and STLs from Onshape

Assuming you have a compatible printer, the easiest way to print is to download the gcode zip file and use the gcodes files found inside. Alternatively, you can create your own gcodes starting from the STL files inside the same zip files or that can be exported from onshape. Onshape always has the latest developmental version but sometimes things can be work in progress. We also recommend using A4 mirror paper like this one to cover the lightbox. It will create more homogenous and stronger IR illumination. A PDF template to print on the BACK of the mirror paper is found in the zip file.
